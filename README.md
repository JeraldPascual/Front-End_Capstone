```bash

# install dependencies
npm install

# run locally

```

## AI-Assisted Workflow

This project uses AI tools for scaffolding, iteration, and code critique. Every AI-generated
change is reviewed, tested, and understood before being committed. Commit messages follow the
Conventional Commits format so the process is traceable from initial scaffold to final build.
